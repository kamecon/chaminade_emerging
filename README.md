# chaminade_emerging

Presentación sobre mercados emergentes realizada para el *Curso de formación sobre economía* organizado por el [Colegio Mayor Universitario Chaminade](https://cmuchaminade.com/).

En la elaboración de la presentación se utilizan las librerías [wbstats](https://cran.r-project.org/web/packages/wbstats/vignettes/Using_the_wbstats_package.html) y [IMFData](https://cran.r-project.org/web/packages/IMFData/README.html) que permiten descargar a través de R datos del Banco Mundial y el Fondo Monetario Internacional.
